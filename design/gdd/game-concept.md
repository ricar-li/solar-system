# Game Concept — 太阳系探险 (Solar System Explorer)

## One-liner

iPad 3D 太阳系：拖转视角、缩放、点选行星看科普卡片；公转/自转可暂停与调速。

## Pillars

1. **看得懂** — 卡通比例（非真实比例），轨道与名字可开关。
2. **点得动** — 点击星球 / 底部快捷条打开中文科普卡片。
3. **零素材** — 贴图、星空、标签全部程序化生成。

## Core Loop

浏览轨道 → 点选行星 → 读科普与趣闻 → 下一颗 / 调速继续观察。

## Modes (v1.1+)

- **Explore**（默认）：自由观察 + 信息卡。
- **Quiz**（猜猜看）：根据趣闻/数据猜行星，强化记忆。

## Out of scope

真实物理比例默认模式、VR、AR、联网内容、真实 NASA 贴图依赖。
