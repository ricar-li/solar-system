# v1.1 完善规格 — 太阳系探险

日期：2026-07-15 · Game Studio Production 阶段增量

## 目标

在不改核心数据模型的前提下，提升「可玩性 / 教育闭环 / 镜头体验」。

## 功能清单

### 1. 选中镜头飞向 (P0)
- 选中行星（点选或底部条）时，相机平滑飞到合适观察距离（约 `displayRadius * 6 + 3`）。
- 提供「重置视角」回到默认全景。

### 2. 小行星带 (P1)
- 在火星与木星轨道之间程序化撒一圈小石块（~180 点），慢公转，增强空间层次。

### 3. 猜猜看 Quiz (P0 教育)
- 控制栏增加「猜猜看」入口。
- 随机一题：展示一句 funFact 或数据，三选一行星名；正确/错误反馈 + 下一题。

### 4. 信息卡导航 (P1)
- 「上一颗」+「下一颗」双向浏览。
- 双击空白处重置视角（与按钮一致）。

### 5. 文档
- 根目录 README 运行说明。
- Game Studio stage/production 脚手架已落地。
