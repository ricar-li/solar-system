## 1. Sprint Goal

在不改核心数据模型的前提下，完成 v1.1「镜头飞向 / 重置视角 / 小行星带 / 猜猜看 / 信息卡双向导航 / README」，打通可玩性与教育闭环。

---

## 2. Acceptance Criteria Checklist

对照 [docs/superpowers/specs/2026-07-15-v1.1-polish.md](../../docs/superpowers/specs/2026-07-15-v1.1-polish.md)

### 2.1 选中镜头飞向 (P0)
- [x] 点选 3D 场景中的任意天体 → 相机平滑飞到观察距离（约 `displayRadius * 6 + 3`）
- [x] 点底部快捷条选中行星 → 同样触发飞向动画
- [x] 动画可见、不跳切；切换另一颗时从当前位置重新飞向
- [x] 太阳也可被选中并飞向

### 2.2 重置视角 (P0)
- [x] 有明确「重置视角」入口（按钮或等价控件）
- [x] 重置后恢复默认全景（约 distance 24、俯视全系统）
- [x] 双击空白处与按钮行为一致
- [x] 重置后信息卡可保持或按设计关闭（行为一致即可）

### 2.3 小行星带 (P1)
- [x] 火星与木星轨道之间可见一圈程序化小石块（约 ~180 点）
- [x] 慢公转；不抢主视觉（体量/亮度低于行星）
- [x] 开关轨道/标签不影响小行星带合理显示
- [x] 不阻挡点击选中行星

### 2.4 猜猜看 Quiz (P0 教育)
- [x] 控制栏有「猜猜看」入口
- [x] 随机出题：funFact / blurb / 直径 / 顺序 / 温度等提示
- [x] 三选一行星名；正确反馈 + 错误反馈
- [x] 「下一题」可连续作答；连续两题尽量不重复同一答案
- [ ] 退出 Quiz 回到 Explore，场景状态不崩溃

### 2.5 信息卡导航 (P1)
- [ ] 「上一颗」可用，顺序与 `SolarData.all` 一致
- [ ] 「下一颗」可用（现有能力保留）
- [ ] 首尾边界：循环或禁用按钮，行为明确无崩溃
- [ ] 切换时信息卡内容与选中高亮同步；相机飞向同步

### 2.6 文档 / 构建
- [ ] 根目录 README：如何用 xcodegen / Xcode / 模拟器运行
- [ ] `xcodebuild` 针对 iPad 模拟器编译通过
- [ ] 不修改核心 `Planet` 数据模型字段语义（仅 UI/场景增量）

## 3. Manual Playtest — iPad Simulator（≤10 步）

**Precondition**: iPad 模拟器（如 iPad Pro 11"）安装最新 Debug 构建；横竖屏均可，优先横屏。

| # | 步骤 | 期望 |
|---|------|------|
| 1 | 启动 App，等待 3D 场景加载 | 见太阳 + 八大行星 + 轨道/标签；帧率稳定 |
| 2 | 拖转视角、双指缩放 | 可旋转/缩放，无抖动崩溃 |
| 3 | 点底部「地球」 | 信息卡打开；相机飞向地球；底部高亮 |
| 4 | 信息卡点「下一颗」→「上一颗」 | 按顺序切换；卡片与镜头同步 |
| 5 | 点 3D 场景中的土星（或点选另一颗） | 飞向该星；选中环/高亮正确 |
| 6 | 点「重置视角」或双击空白 | 回到全景默认视角 |
| 7 | 观察火星–木星之间 | 小行星带可见、慢转、不抢戏 |
| 8 | 打开「猜猜看」，答对一题 | 正确反馈；可下一题 |
| 9 | 故意答错一题后继续 | 错误反馈；可继续；退出回 Explore |
| 10 | 调速 / 暂停 / 轨道·标签开关后重复 3–6 | 控制项与镜头/选中不冲突 |

**Pass criteria**: 上表全部期望成立 + `xcodebuild` 模拟器通过。

## 4. Risks / Out of Scope

| 类型 | 项 |
|------|-----|
| Risk | 相机飞向与手动手势冲突（飞行动画中 pan/pinch） |
| Risk | 小行星带节点过多导致 iPad 模拟器掉帧 |
| Risk | Quiz 题面剧透当前选中行星（需确认是否刻意） |
| Risk | 信息卡首尾循环与 `SolarData.all`（含太阳）顺序不一致 |
| Out | 真实物理比例模式、NASA 真实贴图、AR/VR、联网内容 |
| Out | 新星球/数据字段大改、音效配乐、成就系统 |
| Out | 完整自动化 UI 测试套件（本 sprint 以手动 + 编译门禁为主） |

**Current gap note（brownfield 快照）**: `QuizEngine` 已有；`InfoCard` 仅有「下一颗」；镜头飞向 / 重置 / 小行星带 / Quiz UI / README 仍待落地或未接线。

## 5. Recommended Next Sprint（≤3）

1. **Quiz 进阶**：连对计数、错题回顾、「再学一遍」跳回对应行星信息卡
